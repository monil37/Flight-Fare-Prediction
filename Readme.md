## Designed a project to predict the fare of flights based on key features that influence the price. The main focus was to understand the Data Science Lifecycle and intergrate it while predicting the price:
<br>

1. Perform Data Cleaning / Data Preparation / Data Pre-processing
2. Data visuaslisation(EDA)
3. Perform feature engineering
    - Feature encoding
    - Checking outliers and impute it
    - Feature selection
4. Build machine leaning model and dump it in pickle format
5. Hypertuning the model along with cross validation
